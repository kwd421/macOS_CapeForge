# Cape Forge 1.0.2

Cape Forge 1.0.2 improves the built-in default cursor preview alignment.

## Highlights

- Recentered the bundled default arrow cursor artwork inside its transparent canvas.
- Fixed the default arrow preview appearing slightly shifted toward the upper-left.
- Kept the cursor image canvas size unchanged so existing preview and export sizing behavior remains stable.

## Notes

- This release only changes the bundled default arrow cursor resource.
- Custom cursor pack import, preview, and `.cape` export behavior are unchanged.
