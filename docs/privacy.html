Privacy Policy

Cursie processes everything locally on your device. The app does not create accounts, does not upload files to external servers, and does not collect personal information for analytics or advertising.

What Cursie accesses

  • Folders explicitly chosen by you for reading cursor files
  • A local applied cursor theme file stored in your Application Support folder after applying system cursors
  • A LaunchAgent plist stored in your LaunchAgents folder to keep the applied cursor theme active across reboots

How Cursie uses that information

  • To read .ani and .cur files you select
  • To preview cursor mappings inside the app
  • To apply the selected cursor theme to macOS system cursors when you tap Apply
  • To remove the cursor agent and restore macOS defaults when you choose Restore Defaults

What Cursie does not do

  • Does not collect personal information
  • Does not track you across apps or websites
  • Does not upload cursor files or any other data to remote servers
  • Does not apply or change system cursors without an explicit action from you

Software Updates

Cursie uses the Sparkle framework to check for new versions. When checking for updates, the current app version is sent to our update server. No personal information, system data, or usage statistics are transmitted.
