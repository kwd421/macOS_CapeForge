Cape Forge Privacy Policy

Cape Forge processes files locally on the user’s device. The app does not create accounts, does not upload files to external servers, and does not collect personal information for analytics or advertising.

Information Cape Forge accesses

  • Folders explicitly chosen by the user for reading cursor files
  • A local applied cursor theme file stored under the user's Application Support folder after applying system cursors
  • A user LaunchAgent plist stored under the user's LaunchAgents folder to keep the applied cursor theme active

How the app uses that information

  • To read .ani and .cur files selected by the user
  • To preview cursor mappings inside the app
  • To apply the selected cursor theme to macOS system cursors when the user chooses Apply
  • To restore default macOS cursors and remove Cape Forge's background cursor agent when the user chooses Restore Defaults

What Cape Forge does not do

  • It does not collect personal information
  • It does not track users across apps or websites
  • It does not upload cursor files or exported files to remote servers
  • It does not apply or restore system cursors without an explicit user action in the app
