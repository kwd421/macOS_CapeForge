Features

Everything you need,
nothing you don't.

Live preview

Every role mapped and visible before you apply — actual size and zoomed, side by side.

One-click apply

Replaces your system cursors instantly. To revert, Cursie opens the right System Settings screen.

Drag & drop

Drop a folder or file straight onto the app. Swap individual roles without starting over.

Auto-update

New versions download and install inside the app. No manual trips to GitHub.

Cursie wouldn't exist without Mousecape by Alex Zielenski — the open‑source cursor manager that pioneered .cape themes on macOS.
