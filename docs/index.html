Cape Forge Support

Preview ANI and CUR cursor packs and apply them to macOS.

Cape Forge is a macOS utility for loading Windows cursor packs, previewing mapped cursor roles, and applying the selected theme directly to system cursors. The app reads only the folders and files you choose.

Privacy Policy Contact Support

What Cape Forge does

  • Reads .ani and .cur files from a folder you choose
  • Maps files to common cursor roles automatically
  • Lets you preview and override individual roles
  • Applies the cursor theme directly and can restore defaults

Need help?

  • Make sure your chosen folder contains .ani or .cur files directly inside it
  • If a role looks wrong, use Change Cursor File… for that role
  • Use Restore Defaults to stop Cape Forge's background cursor agent
