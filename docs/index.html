Dress your Mac cursor in any Windows skin.

Cursie loads .cur and .ani Windows cursor packs, maps roles automatically, and applies the theme directly to macOS system cursors — no App Store, no sandbox.

Download for Mac All Releases Support

What it does

  • Reads .ani and .cur files from any folder
  • Maps files to cursor roles automatically
  • Preview animated cursors before applying
  • Override individual roles manually
  • Applies theme directly to macOS system cursors
  • Restores defaults with one click

Need help?

  • Folder must contain .ani or .cur files directly inside it
  • If a role looks wrong, use Change Cursor File…
  • Pointer color looks off? Use Restore Defaults then reset colors in macOS pointer settings
  • Questions? Email support
