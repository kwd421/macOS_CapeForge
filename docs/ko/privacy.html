Cursie

최종 업데이트: 2026년 6월 10일

개인정보 처리방침

Cursie는 모든 처리를 사용자의 기기 내에서만 수행합니다. 계정을 생성하지 않으며, 외부 서버로 파일을 전송하거나 분석·광고 목적으로 개인정보를 수집하지 않습니다.

Cursie가 접근하는 정보

  • 사용자가 직접 선택한 커서 파일 폴더
  • 시스템 커서 적용 후 Application Support 폴더에 저장되는 로컬 테마 파일
  • 재부팅 후에도 커서 테마를 유지하기 위해 LaunchAgents 폴더에 저장되는 LaunchAgent plist 파일

해당 정보의 사용 목적

  • 사용자가 선택한 .ani, .cur 파일 읽기
  • 앱 내 커서 미리보기 표시
  • 사용자가 적용 버튼을 누를 때 macOS 시스템 커서에 테마 적용
  • 기본값 복구 선택 시 커서 에이전트 제거 및 macOS 기본 커서 복원

Cursie가 하지 않는 것

  • 개인정보를 수집하지 않습니다
  • 앱 또는 웹사이트 간 사용자 추적을 하지 않습니다
  • 커서 파일이나 기타 데이터를 외부 서버로 전송하지 않습니다
  • 사용자의 명시적인 동작 없이 시스템 커서를 변경하지 않습니다
