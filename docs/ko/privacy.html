Cape Forge 개인정보 처리방침

Cape Forge는 사용자가 선택한 파일을 기기 내에서 로컬로 처리합니다. 이 앱은 계정을 만들지 않고, 외부 서버로 파일을 업로드하지 않으며, 광고나 분석을 위한 개인정보를 수집하지 않습니다.

앱이 접근하는 정보

  • 사용자가 직접 선택한 커서 파일 폴더
  • 시스템 커서 적용 후 사용자의 Application Support 폴더에 저장되는 로컬 적용 커서 테마 파일
  • 적용된 커서 테마를 유지하기 위해 사용자의 LaunchAgents 폴더에 저장되는 LaunchAgent plist

정보 사용 목적

  • 사용자가 선택한 .ani, .cur 파일 읽기
  • 앱 안에서 역할별 커서 미리보기 표시
  • 사용자가 Apply를 선택했을 때 선택한 커서 테마를 macOS 시스템 커서에 적용
  • 사용자가 Restore Defaults를 선택했을 때 macOS 기본 커서로 복구하고 Cape Forge 백그라운드 커서 agent 제거

Cape Forge가 하지 않는 일

  • 개인정보를 수집하지 않습니다
  • 앱 또는 웹사이트 간 사용자 추적을 하지 않습니다
  • 커서 파일이나 내보낸 파일을 외부 서버에 업로드하지 않습니다
  • 앱 안에서 사용자가 명시적으로 선택하지 않는 한 시스템 커서를 적용하거나 복구하지 않습니다
