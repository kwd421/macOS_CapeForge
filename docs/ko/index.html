Windows .cur, .ani 커서팩을 불러와 macOS 시스템 커서에 바로 적용합니다.

무료 다운로드 macOS 15 Sequoia 이상 · Apple Silicon & Intel

Mac에서 Windows 커서팩을 사용할 마땅한 방법이 없었습니다.

Cursie는 그 필요에서 시작한 앱입니다. 커서 폴더를 불러오고, 역할 배치를 확인하고, 적용합니다. 그것이 전부입니다.

— kwd421
  • 실시간 미리보기 적용 전 모든 역할을 실제 크기와 확대 화면으로 나란히 확인할 수 있습니다.
  • 한 번에 적용 시스템 커서를 즉시 교체합니다. 버튼 하나로 기본값으로 복구됩니다.
  • 드래그 앤 드롭 폴더나 파일을 앱 위에 바로 드래그할 수 있습니다. 개별 역할 교체도 가능합니다.
  • 자동 업데이트 업데이트가 앱 안에서 설치됩니다. GitHub 릴리즈 페이지를 방문할 필요가 없습니다.

Cursie는 Mousecape 없이는 존재하지 못했을 것입니다. Alex Zielenski가 개발한 이 오픈소스 커서 매니저는 macOS에서 .cape 테마를 처음 개척했습니다. 그 토대 위에 만들어진 앱입니다.
