Windows .cur, .ani 커서팩을 불러와 macOS 시스템 커서에 바로 적용합니다.

무료 다운로드 macOS 15 Sequoia 이상  ·  Apple Silicon & Intel  ·  무료

실시간 미리보기

모든 커서 역할을 실제 크기와 확대 화면으로 나란히 확인합니다. 시스템에 아무것도 바뀌기 전에.

한 번에 적용

시스템 커서를 즉시 교체합니다. 되돌리려면 Cursie가 시스템 설정의 정확한 화면으로 안내합니다.

드래그 앤 드롭

폴더나 파일을 앱 위에 바로 드래그하면 됩니다. 팩 전체를 다시 불러오지 않고 개별 역할만 교체할 수 있습니다.

자동 업데이트

새 버전은 앱 안에서 바로 다운로드되고 설치됩니다. GitHub에 들어갈 필요가 없습니다.

Cursie는 Mousecape 없이는 존재하지 못했을 것입니다 — Alex Zielenski가 개발한 이 오픈소스 커서 매니저는 macOS에서 .cape 테마를 처음 개척했습니다.
