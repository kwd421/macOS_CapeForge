Windows .cur, .ani 커서팩을 불러와 macOS 시스템 커서에 바로 적용합니다.

무료 다운로드 macOS 15 Sequoia 이상 · Apple Silicon & Intel · 무료
  • 01 실시간 미리보기 적용 전 모든 역할을 실제 크기와 확대 화면으로 나란히 확인할 수 있습니다.
  • 02 한 번에 적용 시스템 커서를 즉시 교체합니다. 기본값으로 되돌리려면 Cursie가 해당 시스템 설정 화면으로 안내합니다.
  • 03 드래그 앤 드롭 폴더나 파일을 앱 위에 바로 드래그할 수 있습니다. 개별 역할 교체도 가능합니다.
  • 04 자동 업데이트 업데이트가 앱 안에서 설치됩니다. GitHub 릴리즈 페이지를 방문할 필요가 없습니다.

Cursie는 Mousecape 없이는 존재하지 못했을 것입니다 — Alex Zielenski가 개발한 이 오픈소스 커서 매니저는 macOS에서 .cape 테마를 처음 개척했습니다.
