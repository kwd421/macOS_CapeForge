Mac 커서에 Windows 스킨을 입혀보세요.

Cursie는 .cur, .ani Windows 커서팩을 불러와 역할을 자동 매핑하고, 선택한 테마를 macOS 시스템 커서에 직접 적용합니다. App Store 없이, 샌드박스 없이.

Mac용 다운로드 전체 릴리즈 지원 문의

주요 기능

  • 폴더에서 .ani, .cur 파일 읽기
  • 커서 역할 자동 매핑
  • 적용 전 애니메이션 커서 미리보기
  • 역할별 커서 파일 수동 변경
  • macOS 시스템 커서에 직접 적용
  • 버튼 하나로 기본 커서 복구

도움이 필요할 때

  • 선택한 폴더 바로 아래에 .ani 또는 .cur 파일이 있어야 합니다
  • 역할이 맞지 않으면 커서 파일 변경…으로 직접 지정하세요
  • 포인터 색이 이상하면 Restore Defaults 후 macOS 포인터 설정에서 Reset Colors를 사용하세요
  • 문의는 이메일로 주세요
