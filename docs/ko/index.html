Cape Forge 지원

Windows 커서팩을 미리보고 macOS에 직접 적용하세요.

Cape Forge는 Windows 커서팩을 불러와 역할별 커서를 미리보고, 선택한 테마를 macOS 시스템 커서에 직접 적용하는 유틸리티입니다. 앱은 사용자가 직접 선택한 폴더와 파일만 읽습니다.

개인정보 처리방침 지원 문의

Cape Forge가 하는 일

  • 사용자가 선택한 폴더에서 .ani, .cur 파일 읽기
  • 커서 역할에 맞게 파일 자동 매핑
  • 역할별 미리보기 및 개별 파일 변경
  • 커서 테마 직접 적용 및 기본 커서 복구

도움이 필요할 때

  • 선택한 폴더 바로 아래에 .ani 또는 .cur 파일이 있어야 합니다
  • 특정 역할이 맞지 않으면 커서 파일 변경…으로 직접 지정할 수 있습니다
  • Restore Defaults로 Cape Forge 백그라운드 커서 agent를 멈출 수 있습니다
